<Warning>This feature is only available in the Pro plan.</Warning>

This page is for learning how to execute bulk Google dork searches. If you are new to Google dorks, you can learn more about them on our [Introduction to Google Dorks](https://docs.dorki.io/essentials/ai-dorks) page.

1. Navigate to Bulk Dorks

<img
  className="block dark:hidden"
  src="/images/bulk-dorks/bulk-dorks-page.png"
  alt="Hero Light"
/>
<img
  className="hidden dark:block"
  src="/images/bulk-dorks/bulk-dorks-page.png"
  alt="Hero Dark"
/>

2. Upload a file containing a list of dorks to query

<img
  className="block dark:hidden"
  src="/images/bulk-dorks/upload.png"
  alt="Hero Light"
/>
<img
  className="hidden dark:block"
  src="/images/bulk-dorks/upload.png"
  alt="Hero Dark"
/>

3. Select the file you just uploaded then click Run Dorks. The file name convention is: username-filename-id

<img
  className="block dark:hidden"
  src="/images/bulk-dorks/select-file.png"
  alt="Hero Light"
/>
<img
  className="hidden dark:block"
  src="/images/bulk-dorks/select-file.png"
  alt="Hero Dark"
/>

3. Wait, until the search completes for the results to be displayed, the first search is always the slowest because the results need to be cached in memory.

  className="block dark:hidden"
  src="/images/bulk-dorks/results.png"
